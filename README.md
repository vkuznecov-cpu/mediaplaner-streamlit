# Media Planner

Streamlit-приложение для расчета медиаплана по месяцам с экспортом в Excel-шаблоны.

## Возможности

- пресеты метрик: `E-com` и `DIY`;
- расчет по месяцам и типам РК;
- коэффициенты сезонности: `Спрос`, `AOV`, `Кастомный набор`, `Медийные хвосты`;
- настройка НДС и АК, включая фиксированную АК и шкалу от бюджета;
- блок эластичности метрик к сезонности спроса с пресетами и ручной настройкой;
- сводные таблицы и диаграммы, включая выбор строк для построения графика;
- экспорт в Excel-шаблоны;
- экспорт/импорт состояния проекта через JSON.

## Структура проекта

- `app.py` — основная логика Streamlit-приложения;
- `requirements.txt` — зависимости Python;
- `templates/` — Excel-шаблоны для экспорта;
- `assets/` — изображения и SVG для интерфейса;
- `.streamlit/` — локальная конфигурация Streamlit.

## Требования

- Python 3.10+ (рекомендуется 3.11+);
- зависимости из `requirements.txt`.

## Установка

```powershell
python -m venv .venv
.venv\Scripts\activate
python -m pip install --upgrade pip
pip install -r requirements.txt
```

## Запуск

```powershell
python -m streamlit run app.py
```

Обычно приложение доступно на `http://localhost:8501`.

## Работа с GitHub

Перед публикацией в GitHub достаточно перенести корневую папку проекта целиком. В репозиторий стоит включить:

- `app.py`;
- `README.md`;
- `requirements.txt`;
- папку `assets`;
- папку `templates`.

В репозиторий не стоит включать локальные служебные файлы вроде `__pycache__`, `.venv`, временные Excel-файлы `~$...xlsx` и пользовательские JSON-выгрузки, если они содержат рабочие данные.

## Excel-шаблоны

Актуальные шаблоны должны лежать в папке `templates`:

- `templates/Шаблоны МП.xlsx` (для `E-com`)
- `templates/Шаблоны МП DIY.xlsx` (для `DIY`)

Важно:

- используйте именно эти имена файлов;
- не передавайте временные файлы Excel вида `~$...xlsx`.

## Что передавать на проверку

Передавайте архивом весь проект (целиком), включая:

- `app.py`;
- `requirements.txt`;
- папку `templates` с обоими шаблонами;
- (опционально) тестовый JSON проекта, выгруженный из вкладки `Export/Import`.

## Быстрая проверка перед передачей

1. Запустить приложение: `python -m streamlit run app.py`.
2. Открыть вкладку `Коэффициенты`, создать тестовые наборы `Спрос` и `AOV`.
3. Открыть вкладку `Медиаплан`, выбрать пресет, посчитать тестовый сценарий и проверить блоки `НДС и АК`, `Эластичность` и `TOTAL`.
4. Открыть вкладку `Диаграммы`, проверить сводную таблицу, выбор строк для графика и круговые диаграммы.
5. Проверить экспорт:
   - при пресете `E-com` активна кнопка E-com;
   - при пресете `DIY` активна кнопка DIY.
6. Убедиться, что файл Excel открывается без ошибок формул.
