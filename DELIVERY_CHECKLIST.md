# Чек-лист передачи на проверку

## 1. Что отправить

- архив всей папки проекта `media_planner`;
- внутри архива обязательно:
  - `app.py`
  - `requirements.txt`
  - `templates/Шаблоны МП.xlsx`
  - `templates/Шаблоны МП DIY.xlsx`

## 2. Что проверить у получателя

1. Установка зависимостей:
   - `pip install -r requirements.txt`
2. Запуск:
   - `python -m streamlit run app.py`
3. Проверка экспорта:
   - пресет `E-com` -> экспорт E-com шаблона;
   - пресет `DIY` -> экспорт DIY шаблона.

## 3. Важные примечания

- Не включать временные excel-файлы `~$*.xlsx`.
- Не переименовывать шаблоны в `templates`.
